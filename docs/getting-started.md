# Getting Started

KLE-NG runs entirely in your web browser — no installation required.
Visit [editor.keyboard-tools.xyz](https://editor.keyboard-tools.xyz/) and start creating or editing keyboard layouts immediately.

**Existing KLE users:** Your saved layouts are fully compatible. Import your JSON files and continue where you left off.

## Interface Overview

The editor is divided into several panels:

| Panel | Description |
|-------|-------------|
| **Canvas** | The main editing area where keys are displayed and manipulated |
| **Toolbar** | Top bar with import, export, and tool selection buttons |
| **Key Properties** | Right panel for editing individual key properties (size, color, labels) |
| **Keyboard Metadata** | Right panel for layout-wide settings (name, CSS, VIA metadata) |
| **Summary** | Right panel showing layout statistics |
| **JSON Editor** | Right panel for direct JSON editing |
| **Plate Generator** | Right panel for generating plate files |
| **PCB Generator** | Right panel for generating KiCad PCB projects |

## Quick Start

1. **New layout** — The editor starts with a sample layout. Use the toolbar to import an existing layout or start editing.
2. **Select keys** — Click a key on the canvas to select it. Click and drag to select multiple keys.
3. **Edit properties** — With a key selected, adjust its properties (size, color, labels) in the **Key Properties** panel.
4. **Export** — Click **Export** in the toolbar to download your layout in the desired format.
